Super Snake:

  • Creator, Developer, Programmer
  • January 2022
  • (Unity, C#)
  • Duration: 1 weeks
  • Team Size: 2

Background:

Over winter break I was out of schooling, which meant no coding assignments. It did provide a brief area of relaxation and free time, however I didn't wanna just use this time up while falling behind on my skills.

It had been a while since I last made a 2D unity game, so I thought I would give myself a personal game jam to try to make something interesting in one week. I brainstormed many ideas, and a strong contender was to take a classic game and put a unique spin on it. I thought this would be fun to try, and what game could be more classic than snake?

I created the sprites myself (as one can tell) using Gimp. My friend and I worked on two tracks together for the game, and the rest came down to the programming!

Learning Experiences:

At first I was not sure where to even begin. Coding the snake would be the most logical starting position, but the spawning of food felt like an easier way to begin figuring out the games structure. I knew I needed a grind system because the snake travels in squares, and experimenting with the food made me realize what a powerful tool Mathf.Round can be.

I sanctioned the play space into a grind by allowing the snake and food to only appear on whole number positions using Mathf.Round. After basic input and collider code I had the main mechanics finished, I just needed to add the body. For that I used a List in C#, and stored each new fragment of the snake as an object in the list. Once the snake ate the food all I had to do was push back in fragment in the list (minus the head) and append a new fragment to the missing spot!

The biggest impact from this project was that it strengthened my creative skills. It was the first time I brainstormed an idea for a game and its mechanics and coded them all from scratch. I also became more familiar with data structures and operational functions within C#.

Conclusion:

It was not the hardest nor most impactful project I have worked on, but it was really fun. And honestly, that is what game development is about. It is nice to have the creative freedom to create whatever you want from scratch, and once it is finally made there is this pride within you. YOU built this.

When I created this project I had never done an official game jam before. I was worried how stressful they could be, trying to create a game with such a short amount of time. However I was able to create snake all by myself in a week, and I was able to add special fruits with special powers too! I left this project feeling much more confident in myself and much more open to participating in game jams in the future.
